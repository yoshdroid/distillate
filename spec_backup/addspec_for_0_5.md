# Additional Specification for ver0.5
- ステージデータとして 4を定義する
    - 4は 青い三角で表示される
    - 4は 3と同じ排水機能をもつ
    - 4で取り除かれた水は、その色別に数をカウントする

- ゲームルールを追加し、ステージクリア条件を定義する
    - 新規パラメタ STAGE_GOALを追加する
    - 4で取り除かれた水ブロックの総数が STAGE_GOALに達したとき、画面更新を止めステージクリアのメッセージを表示する
    - 新規パラメタ CLEAR_RATEを追加する
    - ステージクリア時、4で取り除かれた"青い水ブロック"の数 / STAGE_GOALの値を計算する
        - 画面には パーセント表記で 小数点第一位まで表示する
        - これが CLEAR_RATE以上であれば "True Clear"であることを表示する
    - 左クリック入力まで待ち、タイトル画面へ戻る
